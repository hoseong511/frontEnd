## 3. 시작!
  - ! + tab
  - ```<!DOCTYPE html>``` HTML5형식에 맞게 실행하도록 명시
  - 시작태그와 종료태그 (**브라우저가 이해할 수 있는 규칙을 학습**)
  - ```<html></html>``` 문서의 전체 범위
  - ```<head></head>``` 문서의 정보를 나타내는 범위
    웹 브라우저가 해석해야 할 웹페이지의 제목, 설명, 사용할 파일 위치, 스타일(css) 같은, 웹페이지의 정보
  - ```<body></body>``` 문서의 구조를 나타내는 범위
