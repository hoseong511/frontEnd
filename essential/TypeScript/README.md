# TypeScript
- Typed JavaScript
- compiles to plain JavaScript
- Transpile

## Transpile
```sh
# global install!
  npm install typescript -g 
  # transpile on File
    tsc test.ts # transpile -> test.js
  # transpile on Project
    tsc --init # tsconfig.json
    tsc # transpile -> test.js
    
# project install
  cd project
  npm init
  npm install typescript
  npx tsc --init
  npx tsc
  npx tsc -w # watch Mode 
```
